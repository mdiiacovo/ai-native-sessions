The live build

Issue → Pull Request → Shipped. No editor required.

The whole loop runs in a browser using the Copilot coding agent — the most accessible way to show "describe the problem, AI delivers."

1

Write the issue

A real pain point, described clearly. A well-written issue is half the work.

2

Assign it to Copilot

On github.com, set the issue's assignee to Copilot. The coding agent starts working on its own.

3

Review the pull request

Copilot opens a PR with the change. Skim the diff and its explanation.

4

Merge → it ships

Merging auto-deploys via GitHub Pages. Refresh the live app — the feature is there.
